# 📈 我的美股投資中心

專為**國泰複委託美股**設計的個人投資管理儀表板。所有資料只存在你自己的電腦，不外流、不需要券商帳密。

## 怎麼啟動
雙擊資料夾裡的 **`執行.bat`** → 瀏覽器自動打開儀表板。關掉黑色視窗即可停止。
（手動：`python -m streamlit run app.py`）

第一次若缺套件，先執行一次：
```
python -m pip install -r requirements.txt
```

## 六大分頁

| 分頁 | 功能 |
|------|------|
| 🏠 **總覽** | 資產總額、總投入、未實現損益、今日損益、產業圓餅圖、快速健檢提醒 |
| 📇 **我的持股** | 卡片式呈現每檔持股（現價、漲跌、損益）。點「查看詳細 →」進入個股頁 |
| 👀 **追蹤清單** | 想買的股票，顯示現價與「該不該進場」判斷，可設目標買價 |
| 📊 **統計報表** | 歷史已實現損益、年度/季度長條圖、個股損益排行（吃你的 Excel）|
| 🤖 **AI 每日簡報** | Claude AI 幫你盤點整個投資組合、點出風險與機會（需 API 金鑰）|
| ⚙️ **管理資料** | 新增/編輯持股、追蹤清單，匯入歷史 Excel |

### 個股詳細頁（點卡片進入）像看股票 App：
即時價、**盤前/盤後價**、今日高低、52週高低、50/200日均線、本益比、市值、
**分析師目標價與評等**、殖利率、**K線走勢圖（含均線、成本線）**、
你的部位損益、**建議停損價**、**+10%/+20% 目標價**、
**該續抱還是賣出的判斷**、**財報日/除息日/配息日**、相關新聞。

## 怎麼開始用（重要）

因為國泰複委託沒有開放 API，所以**你目前持有什麼要手動輸入一次**（之後買賣再更新）：

1. 進 **⚙️ 管理資料 → 📇 我的持股**
2. 在表格輸入你現在持有的每檔股票，只要 3 個數字：
   - **代號**（如 `NVDA`）
   - **股數**
   - **平均成本**（每股美金）
   - （停損價可留空，系統會建議）
3. 按「💾 儲存持股」

其他全部自動：現價、損益、走勢圖、財報、新聞、AI 分析都不用你管。

**追蹤清單**：到 **⚙️ 管理資料 → 👀 追蹤清單** 加想買的代號，可設目標買價。

**歷史報表**：`history.csv` 已經是你的損益 Excel 轉好的。日後要更新，到
**⚙️ 管理資料 → 📊 匯入歷史** 重新上傳國泰的損益 Excel 即可。

## 🤖 AI 每日簡報（選用）
1. 到 [console.anthropic.com](https://console.anthropic.com) 申請 API 金鑰
2. 在「🤖 AI 每日簡報」分頁展開設定、貼上金鑰存檔
3. 按「產生今日 AI 簡報」，Claude 會分析你的整個組合

> 想「每天早上自動產生」？可用 Windows 工作排程器設定，需要的話再跟我說。
> 不設金鑰也沒關係——「🏠 總覽」的快速健檢已提供規則式提醒。

## 常見問題
- **股價/新聞抓不到？** 需要能上網；Yahoo Finance 為免費延遲行情（約 15 分鐘），偶爾抓不到，按側欄「🔄 重新整理」即可。
- **想要真正的即時（0 秒延遲）行情？** 需付費行情源，可再跟我討論。
- **資料會不會不見？** 都存在這個資料夾（`holdings.csv`、`watchlist.csv`、`history.csv`），可自行複製備份。

⚠️ 本工具僅供個人記帳與參考，所有提醒與 AI 分析皆**不構成投資建議**。
